# Running DLLM
Run dllm repo from `dllm/examples/llada/README.md`. Main file with generation logic is at `dllm/dllm/pipelines/llada/generator.py`.

# Running ESM
Run ESM generation from scratch using `esm/examples/lm-design/README.md`. Run ESM conditional generation from `esm/examples/inverse_folding/README.md`. The main file for the former is `esm/examples/lm-design/lm_design.py`. The main file for the latter is `esm/examples/inverse_folding/sample_sequences.py`. Main file with generation logic is at `esm/esm/inverse_folding/gvp_transformer.py`.
